# マツケン配置管理システム 機能一覧

本システムに実装されている全機能を、操作手順を省いて一覧化したものです。
機能ベースでの要件確認、営業資料、機能説明書、他システムとの比較検討などにお使いください。

詳細な操作方法は [USER_MANUAL.md](USER_MANUAL.md) を参照してください。

---

## 1. 認証・ログイン機能

| # | 機能 | 概要 |
|---|---|---|
| 1.1 | ID・パスワードログイン | ユーザー名とパスワードによる標準ログイン |
| 1.2 | パスワード表示／非表示切替 | ログイン時にパスワードを目視確認できる |
| 1.3 | QRコードログイン | 専用QRをスマホカメラで読み取り自動ログイン |
| 1.4 | ロール別リダイレクト | スタッフはサイネージ、それ以外はカレンダーへ自動遷移 |
| 1.5 | セッション管理 | Cookie ベース、有効期間1週間 |
| 1.6 | ログアウト機能 | サイドバー／モバイルメニューから |
